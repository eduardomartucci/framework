Instalação BOWER

Antes de mais nada você precisa realizar a instalação do Node que instalará também o NPM em sua máquina.
Caso não tenha instalado ainda, basta acessar o nodejs.org, fazer download e instalar. Suave!

Após a instalação, vamos para o terminal/cmd e instalar o bower com o seguinte comando:

# npm install -g bower

O NPM é um gerenciador de pacotes de programas que rodam com o Node, com o comando acima estamos realizando a instalar o bower no nosso computador.
A opção -g que está dizendo para o npm instalar o bower globalmente em nossa máquina definindo todas as variáveis de ambiente para utilizar em todos os projetos.

Dicas:

Iniciar Bower para gerar o arquivo “bower.json”:
# bower init

No nosso caso este arquivo já esta criado na raiz da conta (/home/base/public_html/

Instalar dependencias do projeto definidas no arquivo “bower.json”:
# bower install

Pesquisas bibliotecas:
# bower search bootstrap

Instalação GRUNT

# npm install -g grunt-cli

Instalação dos plug-ins necessários do Grunt

# npm install grunt-contrib-uglify --save-dev

# npm install grunt-contrib-concat --save-dev

# npm install grunt-contrib-cssmin --save-dev

Será criado uma pasta node_modules na raiz do projeto onde foram instalados os plugins do Grunt.

Para projetos com os arquivos "package.json" e "Gruntfile.js" já criados e com todos os passos acima já realizados em outros projetos, basta digitar o comando:

# npm install

Baixando o SB Admin do repostório

# git clone https://github.com/BlackrockDigital/startbootstrap-sb-admin-2.git

Executando o comando acima será baixado a última versão do SB Admin em uma pasta chamada startbootstrap-sb-admin-2 na raiz da conta (fora do public_html).
